Quick Start Guide

Get up and running in minutes.

1. Prerequisites

Ensure you have Python 3.8 or higher installed on your system.

2. Install Dependencies

Market Terminal relies on several powerful libraries. Install them via pip:

pip install pandas yfinance matplotlib supabase

3. Run the Application

Launch the terminal interface:

python app.py
